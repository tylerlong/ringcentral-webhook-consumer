# ringcentral-webhook-consumer

Out-of-box consumer for RingCentral WebHook, mainly for testing and troubleshooting purpose.


## Setup

```
yarn install
yarn run upgrade
```


## Development

```
yarn build:watch
yarn start
```


## Pack

```
yarn dist:dev
```


## Distribution

```
yarn dist
```
